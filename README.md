# FastAPI Learning

A structured learning repository for mastering FastAPI and backend development.

## Structure

- `concepts/` — scratch files and experiments, one folder per phase
- `task-api/` — the build project, a Task/Notes API that grows across all phases
- `fastapi-backend-roadmap.md` — the full learning roadmap

## Phases

| Phase | Topic |
|---|---|
| 0 | Python foundations |
| 1 | Pydantic |
| 2 | FastAPI core |
| 3 | Data layer |
| 4 | Auth & security |
| 5 | Testing |
| 6 | Production |
